The first stage, when the Sender chooses which message to send, is repeated over several rounds. On each round, they will see what the bonus for each option is. They might not always be the same! Then they choose which message to send.

First the Sender learns what the possible bonuses are, Option A and Option B.

Each round is linked to a different receiver. Only one message and receiver choice pair will be randomly selected for payment.
